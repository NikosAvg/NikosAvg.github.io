Random Walk in Python

In mathematics a random walk is a random process that describes a path that consists of a succession of random steps on some mathematical space. The simplest example of a random walk is the random walk of the integer line, with starting point the number 0, and at each step moves +1 or -1 with equal probability. The above example is an one-dimensional random walk and will be the first random walk that we will see in this post.
